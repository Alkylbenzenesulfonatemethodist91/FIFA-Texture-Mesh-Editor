## 🚀 Getting Started

When you open the application, you see the main dashboard. The interface splits into three main areas: the file browser on the left, the preview window in the center, and the property inspector on the right.

To start editing, follow these steps:

1. Click File in the top menu and select Open Project.
2. Navigate to your FIFA or EA FC game directory.
3. Select the file you want to edit, such as a texture or mesh file.
4. Wait for the tool to load the data into the center window.
5. Use the preview window to rotate or zoom in on the texture or model.
6. Use the property inspector to change colors, swap textures, or adjust mesh settings.
7. Click Save once you finish your changes.

## 🎨 Modifying Kits and Player Faces

The tool excels at handling uniform and player data. 

To change a kit:
1. Locate the kit texture file in the sidebar.
2. Right-click the file and select Export to save a copy to your computer.
3. Open the file in an image editor.
4. Make your desired changes and save the file in a compatible format like PNG or DDS.
5. Go back to the editor, right-click the original file, and select Replace.
6. Pick your edited file.

To change a player face:
1. Find the player ID in the database list.
2. Load the associated mesh file.
3. Use the mesh tools to adjust geometry or hair assets.
4. Apply your changes and rebuild the cache before launching the game.

## 🛠️ Troubleshooting Common Issues

If you experience problems, check these solutions:

- Tool does not open: Ensure you installed the latest version of the .NET Desktop Runtime from the official Microsoft website.
- Changes do not appear: Make sure you saved your project and applied the mod to your game launcher.
- Visual glitches: Check that your texture files use the correct resolution. Extremely large textures can cause performance issues or crashes.
- Access denied errors: Run the editor as an administrator by right-clicking the icon and selecting Run as administrator.

## 💡 Best Practices for Users

Backup your game files before you start modding. Keep an original copy of your game data in a separate folder. If a modification causes the game to crash, you can quickly restore your files by copying your backup back into the installation directory. 

Start with small changes, like changing a logo on a banner, before you attempt complex tasks like editing player face meshes. This helps you learn how the software handles different file types.
